# Image Files

Place your photos in the following folders:

## `/hero/`
Background or portrait images for the hero section.

## `/gallery/`
Photos for the polaroid gallery. Name them:
- `photo-1.jpg` through `photo-15.jpg`

The gallery currently expects these images:
- `photo-1.jpg` - Photo stack (back)
- `photo-2.jpg` - Photo stack (front)
- `photo-3.jpg` through `photo-11.jpg` - Gallery polaroids

## `/animals/`
Optional: Custom animal illustrations or icons.

## `/icons/`
Optional: Small icons (heart, flower, necklace, etc.)
